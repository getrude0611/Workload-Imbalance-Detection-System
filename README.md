# Workload-Imbalance-Detection-System
This project analyses IT helpdesk ticket data to detect workload imbalance across agents using both rule-based analytics and machine learning.

# Features
- Workload metrics computation
- Rule-based method
- Machine learning model (Decision Tree)
- Streamlit dashboard for visualisation

# How to Run
1. Install dependencies:
   pip install 
pandas
numpy
scikit-learn
streamlit
matplotlib

2. Run the dashboard:
   streamlit run app/dashboard.py
